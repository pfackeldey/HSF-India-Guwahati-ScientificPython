# hsf-india-IITG-Nov2025

Supporting materials for the HSF-India Scientific Computing Workshop at IIT Guwahati held November 24-28 2025. Find out 
more here: https://indico.cern.ch/event/1580062/timetable/

[![Binder](https://binderhub.ssl-hep.org/badge_logo.svg)](https://binderhub.ssl-hep.org/v2/gh/

## Local setup

First, clone this repository:

```shell
git clone https://github.com/pfackeldey/HSF-India-Guwahati-ScientificPython.git

cd HSF-India-Guwahati-ScientificPython
```

Make sure you've [installed pixi](https://pixi.sh/latest/installation/) on your computer.

Then you can install the environment and start a local JupyterLab session with:

```shell
pixi run start
```
